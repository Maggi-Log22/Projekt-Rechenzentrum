# Clustering Analysis of Physiological Sensor Data

This repository contains the Python code used for the preprocessing and clustering analysis of a multisensor physiological dataset.

## Methods
- Exploratory Data Analysis
- Feature Extraction (30s windows)
- K-Means Clustering
- Gaussian Mixture Models

## Files
- K-Means_All.py – implementation of the K-Means clustering
- GMM_All.py – implementation of the Gaussian Mixture Model
- preprocessing.py – preprocessing and feature extraction
- model_comparison.py – comparison of clustering results
